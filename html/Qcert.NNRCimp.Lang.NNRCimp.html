Module Qcert.NNRCimp.Lang.NNRCimp


NNRCimp is a variant of the named nested relational calculus (NNRC) that is meant to be more imperative in feel. It is used as an intermediate language between NNRC and more imperative / statement oriented backends

    Inductive nnrc_imp_expr :=
    | NNRCimpGetConstant : var -> nnrc_imp_expr (* global variable lookup ($$v) *)
    | NNRCimpVar : var -> nnrc_imp_expr (* local variable lookup ($v) *)
    | NNRCimpConst : data -> nnrc_imp_expr (* constant data (d) *)
    | NNRCimpBinop : binary_op -> nnrc_imp_expr -> nnrc_imp_expr -> nnrc_imp_expr (* binary operator (e₁ ⊠ e₂) *)
    | NNRCimpUnop : unary_op -> nnrc_imp_expr -> nnrc_imp_expr (* unary operator (⊞ e) *)
    | NNRCimpGroupBy : string -> list string -> nnrc_imp_expr -> nnrc_imp_expr (* group by expression (e groupby g fields) -- only in full NNRC *)
    .

    Inductive nnrc_imp_stmt :=
    | NNRCimpSeq : nnrc_imp_stmt -> nnrc_imp_stmt -> nnrc_imp_stmt (* sequence (s₁; s₂]) *)
    | NNRCimpLet : var -> nnrc_imp_expr -> nnrc_imp_stmt -> nnrc_imp_stmt (* variable declaration (var $v (:= e₁)? { s₂ }) *)
    | NNRCimpLetMut : var -> nnrc_imp_stmt -> nnrc_imp_stmt -> nnrc_imp_stmt (* variable declaration (var $v (:= e₁)? { s₂ }) *)
    | NNRCimpLetMutColl : var -> nnrc_imp_stmt -> nnrc_imp_stmt -> nnrc_imp_stmt (* mutable collection declaration (coll $v { s1 }; s2) *)
    | NNRCimpAssign : var -> nnrc_imp_expr -> nnrc_imp_stmt (* variable assignent ($v := e) *)
    | NNRCimpPush : var -> nnrc_imp_expr -> nnrc_imp_stmt (* push item in mutable collection (push e in $v) *)
    | NNRCimpFor : var -> nnrc_imp_expr -> nnrc_imp_stmt -> nnrc_imp_stmt (* for loop (for ($v in e₁) { s₂ }) *)
    | NNRCimpIf : nnrc_imp_expr -> nnrc_imp_stmt -> nnrc_imp_stmt -> nnrc_imp_stmt (* conditional (if e₁ { s₂ } else { s₃ }) *)
    | NNRCimpEither : nnrc_imp_expr (* case expression (either e case left $v₁ { s₁ } case right $v₂ { s₂ }) *)
                      -> var -> nnrc_imp_stmt
                      -> var -> nnrc_imp_stmt -> nnrc_imp_stmt
    .

nnrc_imp is composed of the statement evaluating the query and the variable containing the resulut of the evaluation
