Q*cert

An overview of the Q*cert query compiler.

What is Q*cert?

Q*cert is a query compiler written, and partially proven correct, using the Coq proof assistant. The compiler takes some input query and generates code for execution. It supports several backends (notably: Java, Javascript and Spark). Ultimately, the goal is to prove correct the full compilation pipeline, ensuring that the generated code behaves exactly as expected and returns the result as specified by the query semantics.
